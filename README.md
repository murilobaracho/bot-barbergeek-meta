# Bot BarberGeek Meta 💈🤖

Um assistente virtual automatizado desenvolvido para barbearias, integrado com as plataformas da **Meta** (WhatsApp Cloud API / Messenger). Este projeto foi criado para otimizar o atendimento ao cliente, automatizar agendamentos, enviar lembretes e responder às dúvidas frequentes de forma rápida e eficiente.

---

## 🚀 Funcionalidades

- **Atendimento Automatizado:** Fluxo de conversa inteligente para recepcionar clientes de forma personalizada.
- **Gestão de Agendamentos:** Facilita a marcação, consulta e cancelamento de horários diretamente pelo chat.
- **Lembretes Automáticos:** Envio de notificações para reduzir faltas e atrasos dos clientes.
- **Integração com a Meta:** Desenvolvido seguindo os padrões oficiais das APIs da Meta para garantir estabilidade e escalabilidade.

---

## 🛠️ Pré-requisitos

Antes de configurar o bot, certifique-se de ter:
- [Node.js](https://nodejs.org/) instalado (versão LTS recomendada).
- Uma conta de desenvolvedor no [Meta for Developers](https://developers.facebook.com/).
- Um aplicativo configurado no painel da Meta com acesso à API desejada (ex: WhatsApp Business Cloud API).
- Token de acesso permanente e número de teste configurados.
